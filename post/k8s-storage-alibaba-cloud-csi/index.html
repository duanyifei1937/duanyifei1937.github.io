aliyun nas 作为持久化存储卷使用方式

目前阿里云CSI存储插件支持云盘、OSS、NAS等存储类型，并支持通过StorageClass创建动态存储卷；

Nas-csi支持两种存储挂载方式：subpath + filesystem

subpath用于不同pod共用挂载 ReadWriteMany;

以下以subpath为例；

Install

  1. 环境说明：

    1. Kubeadm: v1.20.5
    2. Alibaba-cloud-csi-driver v1.1.1
    3. Nfs-utils 安装；

  2. RBAC 权限

    1

    k apply -f https://github.com/kubernetes-sigs/alibaba-cloud-csi-driver/blob/master/deploy/rbac.yaml

  3. CSI Driver plugin:

    1

    k apply -f https://github.com/kubernetes-sigs/alibaba-cloud-csi-driver/blob/master/deploy/nas/nas-plugin-dedicated.yaml

  5. 需要上传文件到nas 存储卷可挂载到物理节点；
