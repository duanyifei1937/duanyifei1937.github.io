socker概念、使用；

socket概念

Socket, 又称套接字；Linux跨进程通信（IPC，Inter Process Communication）的一种；

优点：unix domain socket (同主机跨进程通信) + internet domain socket(跨主机进程通信)

  • socket Protocol:

    • Sock_stream(流式)：TCP

socker 实践

客户端主机会为发起连接的进程分配一个随机端口去创建一个socket，而server的进程则会为每个连接创建一个新的socket；

客户端：

​ 由于端口最多只有65535个，其中还有1024个是不准用户程序用的，那么最多只能有64512个并发连接；

​ 网卡可配置多IP，也作为一个因子；

服务端：

​ 内存消耗；

​ 进程级打开文件句柄数限制；
