结构体

[toc]

概念

  • 结构体：一种复合数据类型

// 类型定义
type Person struct { 
	name string
	age  int
	sex  string
}

变量定义、初始化

  • 顺序初始化： 依次将结构体内部所有成员初始化；

    • var man Person = Person["haha", 11, "man"]

  • 指定成员初始化：

    • man := Person{name: "haha", sex: "man"}
    • 未初始化的成员变量，取该数据类型的对应默认值；

结构体赋值、比较、传参

  • 成员变量使用：

    • .

  • 比较：

    • 只能使用==, !=, 不能使用< > <= >=

  • 赋值：

    • 相同结构体(成员个数、类型、顺序一致)可以赋值；

  • 结构体地址：

    • 结构体变量地址 == 结构体首个元素的地址

  • 内存占用大小：

    • unsafe.Sizeof()

  • 传参：将结构体变量的值copy；内存使用大、效率低、几乎不用；

结构体指针使用

  • 指针变量定义和初始化：

    • 顺序初始化： 依次将结构体内部成员初始化；

      var man *Person = &Person("haha", "man", 20)

    • New():

      new(Person)

  • 结构体指针传参： – 传引用

    • 指针在64位操作系统下大小一致 == 8Bytes;

指针做函数参数返回值

  • 不能返回局部变量的地址值；
    • 局部变量保存在栈帧上，函数调用结束后，栈帧释放；局部变量地址不再受系统保护，随时可能分配给其他程序；
  • 可以返回局部变量的值；
